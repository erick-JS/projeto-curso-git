Branch

Branches permitem que diferentes versões do mesmo conteúdo se desenvolvam ao mesmo tempo, porém de forma independente,
enquanto você recombina periodicamente as contribuições de diferentes branches em um processo chamado de "fusão" merging.

Criando branch

  • git branch "nome do branch" - Cria um branch, mas o usuário ainda permanece no branch master.
  • git checkout -b "nome do branch" - Cria um branch, sendo que o usuário passa a estar no novo branch criado.

Navegando entre branches

  • git checkout "nome do branch"

Removendo um branch

  • git branch -D "nome do branch" - O branch escolhido só será apagado quando o usuário estiver em ou outro branch.

Listando branches

  • git branch

Enviando a nova branch para o repositório remoto

  • git push origin "nome da branch"

Unindo as alterações do novo branch com o master

  • git merge "nome do branch"
